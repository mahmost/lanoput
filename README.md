# Lanoput

Change input language of input/textarea automatically to comply with the element lang attribute

Forked from [FarsiType 1.3.6](http://www.farsitype.ir)

## Usage

Set `lang` attribute on input or textarea, then use :

```javascript
lanoput.init();
```

after DOM is loaded
